4 Responsive CSS

Responsive for table

Just write your css, inside the media query like this : 

     @media only screen and (min-width: 480px) and (max-width: 767px) {
     }

Responsive for mobile

Just write your css, inside the media query like this : 

     @media only screen and (max-width: 479px) {
     }
